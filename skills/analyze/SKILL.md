# analyze

データファイルを受け取り、分析→グラフ→示唆までをまとめたレポートを作る。

## 手順

1. **データの確認**: ファイルパスを受け取り、まず head で構造(列・型・行数・欠損)を把握する。文字コードに注意(広告管理画面のCSVはShift_JISが多い)。
2. **分析の目的を確認**: 指定がなければデータから推測して「この観点で見ます」と宣言してから進める(広告データなら CPA/CTR/CVR の推移、クリエイティブ別比較、曜日/時間帯傾向が定番)。
3. **分析の実行**: pandasを使う。依存はプロジェクトを汚さないよう一時実行で:
   ```bash
   uv run --with pandas,matplotlib python <スクリプト>
   ```
   集計は必ずスクリプトで行い、目視の暗算で数値を作らない。
4. **グラフ作成**: matplotlib で主要な発見ごとに1枚(多くても4枚)。日本語ラベルが必要な場合はフォント指定(`japanize-matplotlib` を --with に追加)。グラフ設計のガイドライン(dataviz)が利用可能なら先に読む。
5. **レポート**: 以下の構成のMarkdownを作成し、グラフPNGと同じフォルダに保存:
   - TL;DR(3行以内)
   - 主要な発見(数値の根拠付き)
   - 示唆と推奨アクション(意思決定は人間がする前提で、選択肢+根拠を提示)
   - 分析の前提・限界(期間、欠損、注意点)
6. グラフ画像をReadで開いて崩れ(文字化け・重なり)を確認してから報告する。

## 注意

- 個人情報らしき列(氏名・メール・電話)を見つけたら指摘し、レポートには含めない。
- 元データは変更しない(読み取りのみ)。
